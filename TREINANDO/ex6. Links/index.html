Usando links externos

Para criar um link externo, ou seja, um link que leva o usuário para outro site, usamos a tag <a> (de âncora) e o atributo href para especificar o endereço do site de destino.

Exemplo de link externo: Repositório do curso Git Hub

O atributo target="_blank" é usado para abrir o link em uma nova aba do navegador, permitindo que o usuário mantenha a página original aberta.

se nao coloca o target="_blank", o link será aberto na mesma aba do navegador.

exemplo de link interno: Meu Git Hub

rel external indica ao navegador que o link é externo e pode ser tratado de forma diferente.
